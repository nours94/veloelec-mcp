# VeloElec App ChatGPT

## Objectif
Créer une App ChatGPT officielle pour aider les utilisateurs à choisir un vélo électrique adapté à leur budget, usage, taille, distance et terrain.

## Architecture
- GPT / App ChatGPT : conversation utilisateur
- Serveur MCP : outils intelligents
- API VeloElec : catalogue vélos
- UI App : cartes vélos, photos, comparatifs
